# useActionResult

A hook to retrieve the result of the action. Use this in a route component of a route with an action option.

<Info>
	Remember that a route can always be navigated to without an action. Make sure
	the route component using the action result can also handle an `undefined`
	action result.
</Info>

```javascript
import { useActionResult } from 'react-sprout';

let actionResult = useActionResult();
```

<Properties>
	<Property name="actionResult" type="any">
		The result of the action
	</Property>
</Properties>
