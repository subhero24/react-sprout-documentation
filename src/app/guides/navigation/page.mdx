# Navigation

## Links {{ anchor: false }}

The `Link` component is a more powerful alternative for the default [anchor](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/a) element.

```javascript
import { Link } from 'react-sprout';

function Route() {
	return <Link href="/some/url">navigate</Link>;
}
```

If you need a `Link` with loading states and navigation information, there is a [useLink](/hooks/use-link) hook.

```javascript
import { useLink } from 'react-sprout';

function Route() {
	let [Link, busy, loading, navigations] = useLink();

	return <Link href="/some/url">navigate</Link>;
}
```

## Forms {{ anchor: false }}

The `Form` component is a more powerful alternative for the default [form](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/form) element.

```javascript
import { Form } from 'react-sprout';

function Route() {
	return (
		<Form action="/some/url">
			<button>navigate</button>
		</Form>
	);
}
```

If you need a `Form` with loading states and navigation information, there is a [useForm](/hooks/use-form) hook.

```javascript
import { useForm } from 'react-sprout';

function Route() {
	let [Form, busy, loading, navigations] = useForm();

	return (
		<Form action="/some/url">
			<button>navigate</button>
		</Form>
	);
}
```

## Navigate {{ anchor: false }}

For navigating programmatically, there is a [useNavigate](/hooks/use-navigate) hook.

```javascript
import { useNavigate } from 'react-sprout';

function Route() {
	let [navigate, busy, loading, navigations] = useNavigate();

	useEffect(() => {
		let timer = setTimeout(() => {
			navigate('/some/url');
		}, 5000);

		return () => {
			clearTimeout(timer);
		};
	}, []);
}
```

	If you need to navigate as a response to a user click, using a Link or Form is
